How To

Create New Profile
==================
1. Click [Profile] (top menu)
2. If signed in, log out.
3. Choose a name.
4. Click [Create]
5. Wait a few seconds.

Post New Topic
==============

Set Admin
=========
1. Create an account
2. Go to profile
3. Click "pubkey"
4. Click "txt"
5. Copy the public key
6. Save it to admin.key (above html)
7. Rebuild (./clean-build)
