# REGEX cheatsheet
# ================
# /s = single-line (changes behavior of . metacharacter to match newlines)
# /m = multi-line (changes behavior of ^ and $ to work on lines instead of entire file)
# /g = global (all instances)
# /i = case-insensitive
# /e = eval

# \w word
# \W NOT word
# \s whitespace
# \S NOT whitespace
